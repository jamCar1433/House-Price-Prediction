How to run the House Price Prediction.

Extract the folder first if it is in Zip state.

1. Open the House Price Prediction folder in Visual Studio Code (VS Code).

2. To run the project go to VS Code terminal or press Crtl + Shift + `
and input this.

- Input this first and wait:

	pip install -r requirements.txt

- To run locally in Streamlit input this:

	streamlit run app.py

3. Click the link then input the House Price Category Prediction details.
